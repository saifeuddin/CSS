## Css orphans Property


The orphan property controls the minimum number of lines of a paragraph that can be left on the old page. This property only affects paged media such as print. 

            
            @media print {
                p {
                    orphan : 3;
                }
            }

orphans =

minimum number of lines in a paragraph split on the old page.

windows =

minimum number of lines in a paragraph split on the new page.
