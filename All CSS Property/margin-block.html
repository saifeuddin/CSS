## Css margin-block Property


$$ -- The margin-block shorthand property in CSS that sets an element’s margin-block-start and margin-block-end values,both of which are logical properties. t creates space around the element in the inline direction, which is determined by the element’s writing-mode, direction, and text-orientation.


$$ -- If the writing-mode is set to horizontal-lr, the margin-block property will act just like setting margin-top and margin-bottom.

## Syntax 


                
                .module {
                    margin-block: 20px 40px;
                }

## Value


  • length
  • percentages
