CSS mix-blend-mode Property

Definition and Usage

The mix-blend-mode property specifies how an element's content should blend with its direct parent background.

CSS Syntax 

                        mix-blend-mode:
                        normal|multiply|screen|overlay|darken|lighten|color-dodge|color-burn|difference|exclusion|hue|saturation|color|luminosity;
