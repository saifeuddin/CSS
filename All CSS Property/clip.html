CSS clip Property

Example

                        img {
                            position: absolute;
                            clip: rect(0px, 60px, 200px, 0px);
                        }

Definition and Usage

What happens if an image is larger than its containing element?

The clip property lets you specify a rectangle to clip an absolutely positioned element. The rectangle is specified as four coordinates, all from the top-left corner of the element to be clipped.

Note: The clip property does not work if "overflow:visible". Note: The clip property is deprecated and will be replaced by the clip-path property in the future.

Syntax

                        clip: auto|shape|initial|inherit;

Property Values

  1. auto

    No clipping will be applied. This is default

  2. shape

    Clips an element. The only valid value is: rect (top, right, bottom, left)
