CSS Font- Size Property

Definition and Usage

                        The font-size property sets the size of a font.

CSS Syntax

                        font-size:medium|xx-small|x-small|small|large|x-large|xx-large|smaller|larger|length|initial|inherit;

Property Values

  1. xx-small
  2. x-small
  3. smaller
  4. small
  5. medium
  6. larger
  7. large
  8. x-large
  9. xx-large
  10. length
  11. %
