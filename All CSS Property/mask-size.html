## CSS mask-size Property


$$ -- In CSS, the mask-size property specifies the size of a mask layer image. In many ways, it works very much like the background-size property. 


            .element {
                mask-image: url(star.svg);
                
                mask-position: 20px center;

                mask-size: 200px 100px;
            }
        
        .element {
            mask-image: url(image1.png),
            url(image2.png), url(image3.png);
            mask-size: 100px 100%, auto, contain;
        }

Property Values

    
    
        /* Lengths */
            mask-size: 200px; 
                /* width is 200px and height is auto */
            
            mask-size: 50% 100%; 
                /* width is 50% and height is 100% */
        
        
        /* Keywords */
            mask-size: auto;
            mask-size: contain;
            mask-size: cover;
