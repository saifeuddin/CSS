## CSS perspective-origin Property


The perspective-origin property determines the origin for the perspective property. Think of it as the vanishing point of the current 3D-space. The perspective-origin property doesnâ€™t do anything by itself. It has to be defined on an element along with perspective. 




            
        perspective-origin: x-position
        perspective-origin: x-position y-position
        
        perspective-origin: percentage | length | constants





        element1 { 
            perspective-origin: 25% 75%; 
        }

        .element2 { 
            perspective-origin: 10px 25px;
        }

        .element3 { 
            perspective-origin: left bottom; }
