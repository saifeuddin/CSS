## Css bleed Property


$$ -- The bleed property in CSS is for specifying space outside of the page box boundary when determining the size of a printed page. 

            
            @page :left {
                bleed : 10pt;
            }
            
            @page : top{
                bleed : 10pt;
            }

## Property Values


  • auto

    Default is zero unless a crop value has been set on the marks property which creates a default of 6pt.

  • length

    Sets how far outside the bleed area is in each direction. Negative values are accepted, but there is no standard for how they would be implemented. Note that print size units (i.e. pt, cm and in) are accepted.
