Css appearance Property

Definition and Usage

The appearance property is used to display an element using a "platform-native styling" based on the users "operating systems theme".

The appearance property is used for one of two reasons:
  1. To apply platform-specific styling to an element that does not have it by default.
  2. To remove platform-specific styling to an element that does have it by default.
