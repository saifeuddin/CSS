## Css inline-size Property

## Css block-size Property


$$ -- block / inline-size is a logical property that defines the height / width of an element when the writing-mode is horizontal, or the width / height of the element when the writing-mode is vertical.


Example :

        
        .element {
            block-size: 200px;
            inline-size: 700px;
            writing-mode: vertical-lr;
        }

## Syntax and Values 

            block-size : width;
            inline-size : width;
  • length

  • auto

  • fit-content( 5rem )

    This function allows a container to grow until a desired size, then make the text wrap, effectively clamping the content to the provided size value. It can be used on Grid containers, as well as box sizing.


  • min-content

  • max-content

  • unset
