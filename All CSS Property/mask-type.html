## CSS mask-type Property


$$ -- The mask-type CSS property indicates whether the SVG <mask> element is treated as an alpha mask or a luminance mask. 


            mask{
                mask-type: luminance;
            }

Property Values

    
    
            /* Keyword values */
            mask-type: alpha;
                specifies that the alpha values (alpha channel) of the mask 
                layer image should be used as the mask values.


            mask-type: luminance;
                specifies that the luminance values of the mask image should be used as the mask values.

Example :

Open in firefox 
    
    img {
        display: block;
        width: 100%;
        max-width: 300px;
        margin: auto;
        -webkit-mask-image: url(#mask);
        mask-image: url(#mask);
    }

    mask {
        mask-type: alpha;
    }

    svg {
        display: block;
        width: 0;
        height: 0;
    }
