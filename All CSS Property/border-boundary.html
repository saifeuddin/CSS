## Css border-boundary Property


$$ -- The border-boundary property in CSS set constraints to the bounds of an element that affects how the element’s borders behave. 

            
    .element {
        border-boundary: display; /* sets the boundary to the screen  display */
    }

Syntax

border-boundary: none | parent | display;

## Property Values


  • none

    no boundary is set on the border.

  • parent

    sets the boundary where the element’s area and the border edges of its parent meet.

  • display

    sets the boundary where the element’s area and the border edges of the viewport meet.
