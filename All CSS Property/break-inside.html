CSS break-inside Property

Example

                        @media print {
                            img {
                                display: block;
                                break-inside: avoid;
                            }
                        }

Definition and Usage

The break-inside property specifies whether or not a page break, column break, or region break should occur inside the specified element.

The break-inside property extends then CSS2 page-break-inside property.

With break-inside, you can tell the browser to avoid breaks inside images, code snippets, tables, and lists.



Css Syntax 

break-inside: auto|all|always|avoid|avoid-column|avoid-page|avoid-region|column|left|page|recto|region|right|verso|initial|inherit;


Values

  1. auto

    Automatic page/column/region break after the element

  2. avoid

    Avoid a page/column/region break inside the element

  3. avoid-page

    Avoid a page break after the element

  4. avoid-column

    Avoid a column break after the element

  5. avoid-region

    Avoid a region break after the element
