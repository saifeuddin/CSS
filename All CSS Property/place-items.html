## Css place-content Property


$$ -- The place-content property in CSS is shorthand for the align-content and justify-content properties, combining them into a single declaration in CSS Grid and Flexbox layouts. 

            
        .element {
            display: flex;
            place-content: start space-evenly;
        }



        .item {
            display: grid;
            place-content: self-start center;
        }


        .item {
            align-content: self-start;
            justify-content: self-start;
        }

## Syntax 


 
                    place-content: align-content justify-content

Property Values

            
        /* Positional alignment */

        place-content: center       center;
        place-content: flex-start   flex-start;
        place-content: flex-end     end-start;
        place-content: none         start;
        place-content: none         end;
        place-content: none         left;
        place-content: none         right;



        /* Baseline alignment */
        place-content: baseline     baseline;
        place-content: none         first-baseline;
        place-content: none         last-baseline;



        /* Distributed alignment */
        place-content: stretch          stretch;
