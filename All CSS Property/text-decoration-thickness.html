CSS text-decoration-thickness Property

The text-decoration-thickness property in CSS sets the stroke thickness of the decoration line that is used on text in an element. The text-decoration-line value needs to be either underline, line-through, or overline to reflect the thickness property. 

    
    .text {
        text-decoration-line: underline;
        text-decoration-thickness: 2px;
    }



    
    .link {
        text-decoration: underline solid green 1px;
    }
    
        /* The longhand equivalent */
    .link {
        text-decoration-line: underline;
        text-decoration-style: solid;
        text-decoration-color: green,
        text-decoration-thickness: 1px;

Syntax

    
    auto | from-font | length | percentage

Values

  1. auto

    is the default value and will “skip” underlines/overlines when passing through a character.

  2. from-font

    If the first available font has metrics specifying a preferred thickness, it uses that thickness; otherwise it behaves like the auto value.

  3. length

  4. percentage
