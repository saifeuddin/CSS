CSS vertical-align Property

Definition and Usage

The vertical-align property sets the vertical alignment of an element.

In order for this to work, the elements need to be set along a baseline. As in, inline (e.g. <span>, <img>) or inline-block (e.g. as set by the display property) elements. 

                
        img {
            vertical-align: middle;
        }


Note that vertical-align is useful on table-cell elements as well, aligning the content within them. Sticking to top, middle, and bottom is the best bet though, as the other values have inconsistent cross-browser results.

CSS Syntax 

        vertical-align: baseline|length|sub|super|top|text-top|middle|bottom|text-bottom|initial|inherit;

Property Values

  1. baseline

    The element is aligned with the baseline of the parent. This is default

  2. length

    Raises or lower an element by the specified length. Negative values are allowed.

  3. %

    Raises or lower an element in a percent of the "line-height" property. Negative values are allowed

  4. sub

    The element is aligned with the subscript baseline of the parent

  5. super

    The element is aligned with the superscript baseline of the parent

  6. top

    The element is aligned with the top of the tallest element on the line

  7. text-top

    The element is aligned with the top of the parent element's on the line

  8. middle

    The element is placed in the middle of the parent element

  9. bottom

    The element is aligned with the lowest element on the line

  10. text-bottom

    The element is aligned with the top of the parent element's on the line
