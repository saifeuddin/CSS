## CSS max-block-size Property


$$ -- max-block-size is a logical property in CSS that defines the maximum height of an element when the writing-mode is horizontal, or the maximum width of the element when the writing-mode is vertical. 

            
        .element {
            max-block-size: 500px;
            writing-mode: vertical-lr;
        }

## Property Values 


 
        /* Length values */
            max-block-size: 250px;
            max-block-size: 5rem;
        
        
        /* Percentage values */
            max-block-size: 75%;
        
        
        /* Keyword values */
            max-block-size: auto;
            max-block-size: fit-content(5rem);
            max-block-size: max-content;
            max-block-size: min-content
