## Css all Property


$$ -- initial-letter is a CSS property that selects the first letter of the element where it is applied and specifies the number of lines the letter occupies.

## Syntax and Values


initial-letter: normal | number | integer;

  • normal

    Does not apply a scaling effect on the first letter. This can be useful for resetting the value where one might be inherited from the cascade.

  • number
    How many lines the letter should occupy where negative values are not allowed.

  • integer
    How many lines the letter should sink where negative values are not allowed. This is handy should you need to position the letter lower to accommodate tricky spacing issues but, if not specified, it takes the value of number.


Example :

            
            /* Style that first letter! */

            .subhead {
                initial-letter: 2;
            }
