CSS isolation Property

The isolation property defines whether an element must create a new stacking content.

Note: The isolation property is helpful when used with background-blend-mode or mix-blend-mode.

CSS Syntax

isolation: auto | isolate |initial|inherit;


Values

  1. auto

    Default. A new stacking context is created only if one of the properties applied to the element requires it

  2. isolate

    A new stacking context must be created



Example

                        div {
                            isolation: isolate;
                        }

                        div img {
                            mix-blend-mode: multiply;
                        }

                        div h1 {
                        
                            mix-blend-mode: overlay;
                        }

## Where it doesn’t work


You might expect isolation to isolate elements when background-blend-mode is used, but that’s not the case. Background elements are already isolated by their nature in that they do not blend with the content that is behind them.
