CSS @font-face Rule

Example

                        @font-face {
                        font-family: myFirstFont;
                        src: url(sansation_light.woff);
                        }

Definition and Usage

With the @font-face rule, web designers do not have to use one of the "web-safe" fonts anymore.

In the @font-face rule you must first define a name for the font (e.g. myFirstFont), and then point to the font file.

Syntax

                        @font-face {
                        font-properties
                        }

Property values

  1. font-family
    1. name

      2. Required. Defines the name of the font.

  2. src
    1. URL

      2. Required. Defines the URL(s) where the font should be downloaded from

  3. font-stretch
    1. normal

    2. condensed
    3. condensed-semi
    4. condensed-extra
    5. condensed-ultra

    6. expended
    7. expended-semi
    8. expended-extra
    9. expended-ultra

    Optional. Defines how the font should be stretched. Default value is "normal"

  4. font-style
    1. normal
    2. italic
    3. oblique

    Optional. Defines how the font should be styled. Default value is "normal"

  5. font-weight
    1. normal
    2. bold
    3. 100
    4. 200
    5. 300
    6. 400
    7. 500
    8. 600
    9. 700
    10. 800
    11. 900

      12. Optional. Defines the boldness of the font. Default value is "normal"

  6. unicode-range
    1. unicode-range

      2. Optional. Defines the range of unicode characters the font supports. Default value is "U+0-10FFFF"
