CSS word-break Property

The word-break property specifies how words should break when reaching the end of a line.

CSS Syntax 

                    word-break: normal | break-all | keep-all |break-word | initial | inherit;

Property Values

  1. normal

    Default value.Uses default line brake rules

  2. keep-all

    Word breaks should not be used for Chinese/Japanese/Korean (CJK) text. Non-CJK text behavior is the same as value "normal"

  3. break-all

    To prevent overflow, word may be broken at any character

  4. break-word

    To prevent overflow, word may be broken at arbitrary points

word-break can be used in CJK content, the spec says: “To enable additional break opportunities only in the case of overflow, see overflow-wrap“.

From this, we can surmise that word-break is best used with non-English content that requires specific word-breaking rules, and that might be interspersed with English content.
