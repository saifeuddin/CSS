CSS font-variant-caps Property

Definition and Usage

The font-variant-caps property controls the usage of alternate glyphs for capital letters.


Example

                    p.normal {
                        font-variant-caps: normal;
                    }
                    p.small {
                        font-variant-caps: small-caps;
                    }
                    p.all-small {
                        font-variant-caps: all-small-caps;
                    }
                    p.petite {
                        font-variant-caps: petite-caps;
                    }
                    p.all-petite {
                        font-variant-caps: all-petite-caps;
                    }
                    p.unicase {
                        font-variant-caps: unicase;
                    }
                    p.titling {
                        font-variant-caps: titling-caps;
                    }

CSS Syntax

                    font-variant-caps:
                    normal | small-caps| all-small-caps | petite-caps | all-petite-caps | unicase |titling-caps   |initial|inherit|unset;

Property Values

  1. normal

  2. small-caps

    Displays small caps capitals

  3. all-small-caps

    Displays small capitals for both upper and lowercase letters

  4. petite-caps

    Displays petite caps capitals

  5. all-small-caps

    Displays petite capitals for both upper and lowercase letters

  6. unicase

    Displays a mix of small capitals for uppercase letters with normal lowercase letters

  7. titling-caps

    Displays titling capitals
