## CSS text-stroke Property


$$ -- text-stroke is an experimental property that provides text decoration options similar to those found in Adobe Illustrator or other vector drawing applications.


The text-stroke property is actually shorthand for two other properties:
  1. text-stroke-width :

    which takes unit value (1px, 0.125em, 4in, etcetera) and describes the thickness of the stroke effect.

  2. text-stroke-color :

    which takes a color value (hex, rgb/rgba, hsl/hsla, etcetera)


text-stroke also has a companion property, text-fill-color, which will override the color property, allowing for a graceful fallback to a different text color in browsers that do not support text-stroke.

Syntax

                    
        text-stroke : width color;

## Values


  • width
  • color


Example :

                
        h1 {
            text-stroke: 2px red;
        }
