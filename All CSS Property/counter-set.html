## Css counter-set Property


$$ -- The counter-set CSS property, true to its name, sets the starting value for a CSS counter.

Example

            .article {
                counter-reset : section;
            }

            .article section h1:first-of-type::before {
                    counter-set: section 100;
            }

            .article section h1{
                counter-reset: section2;
            }

            .article section h1::before{
                counter-increment: section;

                content: counter(section) '.'
                counter(section2) ' :';
            }

            .article section h2::before{
                counter-increment: section2;

                content: counter(section)"."
                counter(section2) ' : ';
            }
