CSS background-size Property

Definition and Usages

The background-size property specifies the size of the background images.

There are four different syntaxes you can use with this property:
the keyword syntax ("auto","cover" and "contain"),
the one-value syntax (sets the width of the image (height becomes "auto"),
the two-value syntax (first value: width of the image, second value: height),
and the multiple background syntax (separated with comma).

Css Syntax 

                    background-size: auto|length|cover|contain|initial|inherit;

Property Values

  • auto

    • Default value. The background image is displayed in its original size

  • length

    • Sets the width and height of the background image.
    The first value sets the width, the second value sets the height.
    If only one value is given, the second is set to "auto".

  • percentage

    • Sets the width and height of the background image in percent of the parent element.
    The first value sets the width, the second value sets the height.
    If only one value is given,the second is set to "auto"

  • cover

    • Resize the background image to cover the entire container, even if it has to stretch the image or cut a little bit off one of the edges

    Its Responsive

  • contain

    • Resize the background image to make sure the image is fully visible
