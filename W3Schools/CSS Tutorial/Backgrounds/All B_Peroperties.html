Css Backgrounds

Backgrounds properties are used to add background effects for elements

Css Background Properties

  1. background-color

    2. Sets the background color of an element

  2. background-image

    3. Sets the background image for an element

  3. background-repeat

    4. Sets how a background image will be repeated

  4. background-attachment

    5. Sets whether a background image is fixed or scrolls with the rest of the page

  5. background-position

    6. Sets the starting position of a background image

  6. background (Shorthand for all Properties)

    7. Sets all the background properties in one declaration



  7. background- clip

    8. Specifies the painting area of the background

  8. background- origin

    9. Specifies where the background image(s) is/are positioned

  9. background- size

    10. Specifies the size of the background image(s)
