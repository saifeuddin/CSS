Position The List Item Markers




List Style Position Property

The list-style-position property specifies the position of the list-item markers (bullet points).

Values

  1. list-style-position: outside

    2. "list-style-position: outside;" means that the bullet points will be outside the list item. The start of each line of a list item will be aligned vertically. This is default:

  2. list-style-position: inside

    3. "list-style-position: inside;" means that the bullet points will be inside the list item. As it is part of the list item, it will be part of the text and push the text at the start:
