CSS Combinators

Combinator is something that explains the relationship between the selectors

A css selector can contain more than one simple selector. between the simple selectors, we can include a combinator

  1. Descendant Selector (space)
  2. Child Selector (>)
  3. Adjacent Sibling Selector (+)
  4. General Sibling Selector (~)






All Css Combinator Selectors

  1. element element

    2. div p

    Selects all p elements inside a div

  2. element > element

    3. div > p

    Selects all direct child p elements where the parent is a div element

  3. element + element

    4. div + p

    Selects the first p element that are placed immediately after div element

  4. element ~ element

    5. p ~ ul

    Selects every ul element that are preceded by a p element
