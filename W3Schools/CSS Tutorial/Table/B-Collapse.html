CSS border-collapse Property

Definition and Usage

The border-collapse property sets whether table borders should collapse into a single border or be separated as in standard HTML.

Css Syntax 

            border-collapse: separate(default) |collapse|initial|inherit;

Property values

  1. separate

    2. Borders are separated; each cell will display its own borders. This is default.

  2. collapse

    3. Borders are collapsed into a single border when possible (border-spacing and empty-cells properties have no effect)
