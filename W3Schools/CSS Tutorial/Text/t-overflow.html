CSS text-overflow Property

Definition and Usage

The text-overflow property specifies how overflowed content that is not displayed should be signaled to the user.

It can be clipped, display an ellipsis (...), or display a custom string.

Both of the following properties are required for text-overflow:

  1. white-space: nowrap;
  2. overflow: hidden;

Css Syntax 

                    text-overflow: clip|ellipsis|string|initial|inherit;

Property Values

  1. clip

    2. Default value. The text is clipped and not accessible

  2. ellipsis

    3. Render an ellipsis ("...") to represent the clipped text

  3. string

    4. Render the given string to represent the clipped text

    Note The text-overflow: "string" only works in Firefox.
