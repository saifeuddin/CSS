CSS display Property

Definition and Usage

The display property specifies if/how an element is displayed.

The display property specifies the display behavior (the type of rendering box) of an element.

In HTML , the default display property value is taken from the HTML specifications or from the browser/user default style sheet. The default value in XML is inline, including SVG elements.



Property values

  1. none

    2. The element is completely removed


  2. block

    3. Displays an element as a block element (like p). It starts on a new line, and takes up the whole width

  3. inline-block

    4. Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values


  4. inline

    5. Displays an element as an inline element (like span tag). Any height and width properties will have no effect


  5. contents

    6. Makes the container disappear, making the child elements children of the element the next level up in the DOM


  6. flex

    7. Displays an element as a block-level flex container

  7. inline-flex

    8. Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values


  8. grid

    9. Displays an element as a block-level grid container

  9. inline-grid

    10. Displays an element as an inline-level grid container


    1. table

      2. Let the element behave like a table element

    2. inline-table

      3. The element is displayed as an inline-level table.


    3. table-caption

    4. table-header-group

      5. Let the element behave like a thead element

    5. table-footer-group

      6. Let the element behave like a tfoot element


    6. table-row

      7. Let the element behave like a tr element

    7. table-row-group

      8. Let the element behave like a tbody element


    8. table-column

      9. Let the element behave like a col element

    9. table-column-group

      10. Let the element behave like a colgroup element


    10. table-cell

      11. Let the element behave like a td element


  10. list-item

    11. Let the element behave like a li element

  11. run-in

    12. Displays an element as either block or inline, depending on context
