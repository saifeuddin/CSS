CSS Font Size




Font Size

The font-size property sets the size of the text.

font-size value can be absolute or relative size.



Absolute Size

  1. Set the text to a specific size
  2. Absolute size is useful when physical size of the output is known
  3. Does not allow a user to change text size in all browsers(bad accessibility reasons)

Relative Size

  1. Sets the size relative to surrounding elements
  2. Allow a user to change text size in browsers



Set fonts with Pixels

Setting text size with pixels gives you full control over text size.

Tip If you use pixels, you can still use the zoom tool to resize tha entire page.




Set Font Size with Em

To allow users to resize the text.




Responsive Font Size

Text size can set with a vw unit, which means the "viewport width"

That way text size will follow the size of the browser window.
