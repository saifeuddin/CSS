Css Borders

The CSS border properties allow you to specify the style, width, and color of an element's border.

CSS Border Style

The border-style property specifies what kind of border to display.

The following values are allowed:

  1. dotted
  2. solid
  3. double
  4. grove

    5. - Defines a 3D grooved border. The effect depends on the border-color value.

  5. ridge

    6. - Defines a 3D ridged border. The effect depends on the border-color value.

  6. inset

    7. - Defines a 3D inset border. The effect depends on the border-color value.

  7. outset

    8. - Defines a 3D outset border. The effect depends on the border-color value.

  8. none
  9. hide




The border-style property can have from one to four values (for the top border, right border, bottom border, and the left border).

Example: 

                        p.mix {
                            border-style: dotted dashed solid double;
                        }
