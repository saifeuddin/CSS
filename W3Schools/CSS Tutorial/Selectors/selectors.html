CSS Selectors

A CSS selector selects the HTML element(s) you want to style .

Css Selectors

We can divide Css Selectors into Five Categories

  1. Simple Selectors

    2. Select elements based on name, id, class

  2. Combinator Selectors

    3. Select elements based on specific relationship between them

  3. Pseudo-class Selectors

    4. Select element based on a certain stage

  4. Pseudo-element Selectors

    5. Select and style a part of an element

  5. Attribute a Selectors

    6. Select elements based on attribute or attribute value

The CSS element Selector

The element selector selects HTML elements based on the element name.

Example:

Here, all elements on the page will be center-aligned, with a red text color:

p { text-align: center; color: red; }

The CSS id Selector

The id selector uses the id attribute of an HTML element to select a specific element.

Example:

The CSS rule below will be applied to the HTML element with id="para1": 

#para1 {
          text-align: center;
          color: red;
        }

The Css class Selector

Class selector selects html elements with a specific class attribute .

To select elements with specific class, write a period(.) character, followed by the class name .

Example: 

 .center {
        text-align: center;
        color: red;
        }

Example:

In this example only p elements with class="center" will be red and center-aligned: 

 p.center {
        text-align: center;
        color: red;
        }

The Universal Selectors

* Selects all html elements on the page

Example:

The CSS rule below will affect every HTML element on the page:

 * {
        text-align: center;
        color: blue;
        e}

The Css Grouping Selector

The grouping selector selects all the HTML elements with the same style definitions.

Example:

In this example we have grouped the selectors from the code above:

 
            h1, h2, p {
            text-align: center;
            color: red;
            }
