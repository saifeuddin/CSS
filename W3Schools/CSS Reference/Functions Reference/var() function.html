CSS var() Function

Example

                        :root {
                        --main-bg-color: coral;
                        }

                        #div1 {
                        background-color: var(--main-bg-color);
                        }

                        #div2 {
                        background-color: var(--main-bg-color);
                        }

Definition and Usage

The var() function is used to insert th a value of a Css variable.
