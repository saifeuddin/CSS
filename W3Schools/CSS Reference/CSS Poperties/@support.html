CSS @support Selector

What Is @support Selector?

Feature Detecter

It is used to check property browser support. Is also called Conditional Rule.

Syntax 

                        #main {
                            display: block;
                        }
                        @support (display:grid){
                            #main {
                                display: gird;
                            }
                        }
                        }
                        @support not (display:grid){
                            #main {
                                display: block;
                            }
                        }




                        @support (display:flex) or (display:-webkit-flex) {
                            #main {
                                display: flex;
                            }
                        }
                        @support (display:flex) and (display:-webkit-flex) {
                            #main {
                                display: flex;
                            }
                        }
                        @support (display:flex) and ( not (display:-webkit-flex)) {
                            #main {
                                display: flex;
                            }
                        }
