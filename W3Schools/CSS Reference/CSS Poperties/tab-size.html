CSS tab-size Property

Definition and Usage

The tab-size property specifies the width of a tab character.

In HTML, the tab character is usually displayed as a single space-character, except for some elements, like 

  and 
 , and the result of the tab-size property will only be visible for those elements.
            

        
CSS Syntax

            
                tab-size: number|length|initial|inherit;
            


        
Property Values 

        
    
            
  1. number
                
    
                    The number of space-characters to be displayed for each tab-character. Default value is 8
