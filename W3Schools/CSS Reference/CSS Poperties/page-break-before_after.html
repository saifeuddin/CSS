CSS page-break-before Property

Example

Always insert a page-break before a h1 element: 

                        @media print {
                        h1 {page-break-before: always;}
                        }

Definition and Usage

The page-break-before property adds a page-break before a specified element..

Tip: The properties: page-break-before, page-break-after and page-break-inside help to define how a document should behave when printed. Note: You cannot use this property on an empty "div or on absolutely" positioned elements.

Syntax 

                        page-break-before: auto|always|avoid|left|right|initial|inherit;

Values

  1. auto

    Default. Automatic page breaks

  2. always

    Always insert a page-break before the element

  3. avoid

    Avoid page-break before the element (if possible)

  4. left

    Insert page-break before the element so that the next page is formatted as a left page

  5. right

    Insert page-break before the element so that the next page is formatted as a right page
