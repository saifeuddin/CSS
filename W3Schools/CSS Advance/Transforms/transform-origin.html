CSS transform-origin Property

Definition and Usage

The transform-origin property allows you to change the position of transformed elements.

2D transformations can change the x- and y-axis of an element. 3D transformations can also change the z-axis of an element.

Note: This property must be used together with the transform property.

Default value: 50% 50% 0

Syntax

                        transform-origin: x-axis y-axis z-axis|initial|inherit;

Property Values

  1. x-axis

    Defines where the view is placed at the x-axis. Possible values:

    • left
    • right
    • center
    • length
    • %

  2. y-axis

    Defines where the view is placed at the y-axis. Possible values:

    • top
    • center
    • bottom
    • length
    • %

  3. z-axis

    Defines where the view is placed at the z-axis(for 3D transforms) . Possible values:

    • length
