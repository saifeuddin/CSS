CSS border-image-width Property


Definition and Usage

The border-image-width property specifies the width of the border image.


Property Values

  • Length

    A length unit (px) specifying the size of the border-width

  • Number

    Default value 1. Represents multiples of the corresponding border-width. Means if value increments width will increment.

  • %

    Refers to the border image area: the width of the area for horizontal offsets, the height for vertical offsets.

  • auto

    If specified, The width is intrinsic(অভ্যন্তরীণ) width or height of the corresponding image slice



Note: The border-image-width property can take from one to four values (top, right, bottom, and left sides). If the fourth value is omitted, it is the same as the second. If the third one is also omitted, it is the same as the first. If the second one is also omitted, it is the same as the first.
