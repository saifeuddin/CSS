CSS Transitions

CSS transitions allows you to change property values smoothly, over a given duration.

CSS Transitions Properties

  1. transition
  2. transition-duration
  3. transition-property
  4. transition-delay
  5. transition-timing-function

How to Use CSS Transition ?

To create a transition effect, you must specify two things

  1. Css property u want to add an effect
  2. Duration of the effect
    3.                         div{
                        transition: width 2s;
                        }

    Note:

    If duration property is not specified, transition will not work




Change Several Property Values

Example:

                        div {
                        transition: width 2s, height 4s;
                        }



Delay the Transition Effect

The transition-delay property specifies a delay (in seconds) for the transition effect.

Example:

                        div {
                        transition-delay: 1s;
                        }



Specify the Speed Curve of the Transition

The transition-timing-function property specifies the speed curve of the transition effect.

Property Values

  1. ease

    slow start, then fast, then end slowly (this is default)

  2. ease-in

    Slow Start

  3. ease-out

    Slow End

  4. ease-in-out

    Slow Start & End

  5. linear

    Always Same

  6. cubic-bezier(n,n,n,n)



Transition + Transformation

Example:

                        div {
                        transition: width 2s, height 2s, transform 2s;
                        }
