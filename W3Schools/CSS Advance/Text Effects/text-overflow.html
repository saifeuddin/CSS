CSS Text Overflow

The CSS text-overflow property specifies how overflowed content that is not displayed should be signaled to the user.

It can be clipped:

or it can be rendered as an ellipsis (...):

Both of the following properties are required for text-overflow:

  • white-space:nowrap
  • overflow:hidden

Property Value

  1. ellipsis

    Render an ellipsis ("...") to represent the clipped text

  2. clip

    Default value. The text is clipped and not accessible

  3. string

    Render the given string to represent the clipped text

    Same as clip
