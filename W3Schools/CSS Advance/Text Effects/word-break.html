CSS word-break Property


Definition and Usage

The word-break property specifies how words should break when reaching the end of a line.

Property Values

  1. normal(default)
  2. break-all

    To prevent overflow, word may be broken at any character

  3. keep-all

    Word breaks should not be used for Chinese/Japanese/Korean (CJK) text. Same as value "normal"

  4. break-word

    To prevent overflow, word may be broken at arbitrary points(ইচ্ছামত বিন্দুতে)
