CSS is() Pseudo Class Selector

The pseudo-select :is() in CSS allows you to write compound selectors more tersely. For example, rather than writing: 

    
    
    ul li,
    ol li {}
    
    

    
We could write:
    :is(ul, ol) li {}




Method 1

    
    
    :is(section, article) h1 {
        
    }

Method 2

    
    
    :is(section, article, aside, nav) :is(h1, h2, h3, h4, h5, h6) {
    color: #BADA55;
    }

Method 3

    
    
    header:is(.back, .header) {
    color: #BADA55;
    }

Method 4

    
    
    :is(.div1, .div2) p:hover {
        font-family: arial;
        color: rgb(189, 183, 194);
    }

Method 5

    
    
    :is(.div1, .div2):hover :is(p) {
        font-family: arial;
        color: blueviolet;
    }
