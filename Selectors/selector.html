Basic Type Selectors

  1. Name : universal Selector
    Css : *
    # Selects All Elements,Every think,Every Single thing(Except ::before,::after).

  2. Name : Type/Element Selector
    Css : div
    # Select elements of that Type.

  3. Name : Class Selector
    Css : .class name
    # Select elements with that Class.

  4. Name : Id Selector
    Css : #id name
    # Selects element with that Id.


    Combination Selectors

  1. Name : Descendant(বংশধর) Selector
    Css : div a
    # Select elements that are "descendants" of the first element.

    Select all anchors(a) that are inside a div (Direct Child & Indirect Child all will selected).


  2. Name : Or Selector
    Css :div, p
    Css :div, p.lorem
    # Select elements that match any selector in "the list" .

    Selects all anchors and all divs.


  3. Name : And Selector
    Css : div.c
    # Select elements that match all the selector "combinations".

    Selects all divs with the class c.


  4. Name : Direct Child
    Css : div > a
    # Selects elements that are "direct children" of the first element.

    Select anchors that are direct children of a div ("Not indirect" only Direct Children).


  5. Name : General Siblings
    Css : li.red ~ li
    # Select elements that are siblings of the first element and come after the first element.

    Selects all anchors that are siblings of a div and come after the div.


  6. Name: Adjacent Siblings
    Css : div .c + a
    # Select elements that are "siblings of the first element" and come "directly after" the first element.

    Selects all anchors that are siblings of a div .c and come "directly after" the div .c .


    Pseudo Class Selectors state

  1. Name : Hover Selector
    Css : button:hover
    # Select elements that are hovered by the mouse.
    Select buttons that are being hovered.

  2. Name : Focus Selector
    Css : button:focus
    # Select elements that are focused. Select buttons that are being focused
    *Focus is set by "either tabbing" to an element or "clicking an element" such as a input, button or anchor tag.

  3. Name : Required Selector
    Css : input:required
    # Select elements that "are required".
    Select inputs with the required attribute.

  4. :optional

  5. Name : Checked Selector
    Css : input:checked
    # Select "checkboxes/radio" buttons that are checked.
    Select inputs that are checked.

  6. Name : Disabled Selector
    Css : input:disabled
    # Select inputs that are disabled
    Select inputs with the disabled attribute



    7. Pseudo class Selector position/other

  7. Name : First Child Selector
    Css : a:first-child
    # Select elements that are the first child "inside a container".
    Select anchors that are the first child.


  8. Name : Last Child Selector
    Css : a:last-child
    # Select elements that are the last child "inside a container".
    Select anchors that are the last child.


  9. Name : Nth Child Selector
    Css : a:nth-child(2n)
    # Select element that are the nth child "inside a container" based on formula.
    Select anchors that are even number children.

  10. Name : Nth Last Child Selector
    Css : a:nth-last-child(3)
    # Select element that are the nth child "inside a container" based on formula "counting from the end".
    Select anchors that are third to last children.

  11. Name : Only child Selector
    Css : p:only-child
    # Select element that are the only child "inside a container".
    Select Select p that are the only child.

  12. Name : First Of type Selector
    Css : a:first-of-type
    # Select elements that are the first of a type "inside/outside" a container.
    Select the first anchor in/outside a container.

  13. Name : Last Of type Selector
    Css : a:last-of-type
    # Select elements that are the last of a type "inside/outside" a container.
    Select the last anchor in/outside a container.

  14. Name : Nth Of Type Selector
    Css : a:nth-of-type(2n)
    # Select elements that are the nth of a type "inside/outside" a container based on the formula.
    Select every second anchor.

  15. Name : Nth Last Of Type Selector
    Css a:nth-last-of-type(2)
    # Select elements that are the nth of a type inside/outside a container based on the formula counting from the end.
    Select the second to last anchor.

  16. Name : only Of Type Selector
    Css : a:only-of-type(n)
    # Select elements that are the only of a type inside/outside a container.
    Select anchors that are the only anchor in/outside a container.


  17. Name : Not Selector
    Css : a:not(.green)
    # Select all elements that do not match the selector inside the not selector.
    Select all anchor tags that do not have the green class.


    Pseudo Element Selectors

  1. Name : Before Selector
    Css : div.red::before
    # Creates an empty element directly before the children of selected element.
  2. Name : After Selector
    Css : div.red::after
    Creates an empty element directly after the children of selected element.



    Attribute Selectors

    The Seven Different Types

  1. Name : Has Attribute Selector
    Css : [href]
    # Select elements that have that attribute.

    Select elements with the href attribute.


  2. Name : Exact Attribute Selector
    Css : [type="text"]
    # Selects elements that have attribute "with exactly that value" .

    Selects elements with type attribute with value text.


  3. Name : Containing a Space-Separated Word Attribute Selector
    Css : [title="flower new"]

    Css : [title ~ ="flower"]
    Css : [title ~ ="new"]

    # Select all elements with a title attribute containing the word "flower"
    # Selects all elements with a title attribute containing the word "new".

  4. Name : Begins/Ends With Hyphen-Separated(-) Attribute Selector
    Css : [type="another-link"]

    Css : [type|="link"]
    Css : [type|="another"]

    # Selects elements that have that attribute which "start/end with that value",value will be hyped separated or any separated two words .

    Selects elements with type attribute with a value ends with link. Selects elements with type attribute with a value starts with another.

  5. Name : Begins With Attribute Selector
    Css : [type^="te"]
    # Selects elements that have that attribute which "start with that value" .

    Selects elements with type attribute with a value starts with te.

  6. Name : Ends With Attribute Selector
    Css : [type$="xt"]
    # Selects elements that have that attribute which "ends with that value" .

    Selects elements with type attribute with a value ends with xt.

  7. Name : Substring Attribute Selector
    Css : [type*="x"]
    # Selects elements that have that attribute "which contain" that "value anywhere" .

    Select elements with the a attribute with a value that contains x.
