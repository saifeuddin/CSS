What is difference between justify-self, justify-items and justify-content in CSS grid?

  • justify-content -

    This property aligns the "Hole Grid " along the Main/Row/Horizontal axis.

  • justify-items -

    Aligns "All Grid Item" inside a container along the Main/Row/Horizontal axis.

  • justify-self -

    Align "Only the Specific Grid Item" along the Main/Row/Horizontal axis inside grid container.

What is difference between align-self, align-items and align-content in CSS grid?

  • align-content -

    This property aligns the "Hole Grid" along the Cross/Column/Vertical axis.

  • align-items -

    Aligns "All Grid Item" inside a grid container along the Cross/Column/Vertical axis.

  • align-self -

    Align "Only the Specific Grid Item" along the Cross/Column/Vertical axis inside grid container.

justify-content

This property controls the "alignment of grid columns".
It is set on the grid container.
It does not apply to set or control the alignment of grid items.

justify-items

This property controls the alignment of grid items.
It is set on the grid container.

justify-self

This property overrides justify-items on individual items.
It is set on grid items and inherits the value of justify-items, by default.
